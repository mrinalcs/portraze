## Usage

1. Clone this repository:

   ```bash
      git clone https://github.com/mrinalcs/portraze.git
   ```
2. Using with Remote Theme Plugin

  In your `_config.yml`, add the following lines

```
plugins:
  - jekyll-remote-theme

  remote_theme: mrinalcs/portraze

```
